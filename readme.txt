== Description ==

Gravity Form Lawmatics Plugin sends form submissions from [Gravity Form](https://www.gravityforms.com/) plugins to Lawmatics when someone submits a form. Learn more about Gravity Form Lawmatics Plugin at [weconnectcode.com](https://www.weconnectcode.com/plugin/lawmatics-for-gravity-forms). We also offer other connector for Clio [Gravity Form Clio Integration](https://www.weconnectcode.com/plugin/clio-grow-for-gravity-forms).

== How to Setup ==

* Go to "Lawmatics Accounts" tab then add new account.
* Go to "Lawmatics Integration" tab then create new Integration.
* Map required Lawmatics fields to gravity form fields.
* Send your test entry to Lawmatics by submiting gravity form.
* Go to "Lawmatics Logs" tab and verify, if entry was sent to Lawmatics.

**Connect Lawmatics account**

Connect any gravity form to Lawmatics account by safe and secure Oauth 2.0.

**Map Lawmatics fields**

First select any Lawmatics object then Map contact form fields to Lawmatics object fields. There is No limitation on number of fields. You can map unlimited fields.

**Filter gravity form submissions**

Either Send all gravity form submissions to Lawmatics or filter contact form submissions sent to Lawmatics based on user input. For example , send only those entries to Lawmatics which have work email address.

**Manually send to Lawmatics**

Send gravity form submissions to Lawmatics when someone submits a contact form. You can manually send contact form submissions to Lawmatics.

**Lawmatics logs**

View a detailed log of each gravity form submission whether sent (or not sent) to Lawmatics and easily resend any contact form submission to Lawmatics.

**Send Data As Lawmatics object Notes**

Send one to many contact form fields as Lawmatics object notes when anyone submits a form on your site.

= How to integrate Lawmatics in gravity form =

You can easily integrate Lawmatics in gravity form. Simply install Gravity Form Lawmatics plugin , Connect Lawmatics account and finally map contact form fields to Lawmatics object(contact,lead,account etc) fields.

= Contact form 7 Lawmatics web to lead =

if API access is not enabled for your organization then simply use Lawmatics web to lead. Enter your organization id and map contact form fields to Lawmatics case or lead fields.

= Wordpress Lawmatics Integration =

* You can use [Gravity Form](https://www.gravityforms.com/) or any other wordpress forms plugin.
* Install this free Wordpress Lawmatics Integration plugin then Connect your Lawmatics account to wordpress.
* Create a Lawmatics Integration , select form then map form fields to Lawmatics Object(Lead, account etc) fields.
* When anyone submits form , it will go to your Lawmatics account.

= Lawmatics Wordpress Plugin =
* Easily Connect your Lawmatics account to wordpress site with this free wordpress Lawmatics plugin.
* First Install [Gravity Form](https://www.gravityforms.com/) plugin and create a simple form.
* Connect Lawmatics account to wordpress.
* Create a Lawmatics Integration then map contact form fields to Lawmatics object(lead, contact etc) fields.
* All form submissions will automatically go to your Lawmatics account.
